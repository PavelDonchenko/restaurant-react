## Getting Started with Create React App
This is my pet project. A restaurant-store. Features a shopping cart, dishes card, sliders.

#Technologies used
* React 18 with hooks(useState, useEffect, useContext)
* React Router
* Slider-Swiper
* Backend - MockApi
* React-content-loader
* Redux-toolkit
